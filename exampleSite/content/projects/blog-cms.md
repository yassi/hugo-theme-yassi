A headless CMS designed specifically for blogs and content-heavy websites.

## Core Features

- Rich text editor with markdown support
- Media library with image optimization
- Custom content types
- Version history and drafts
- SEO optimization tools
- Scheduled publishing
- Multi-author support
- Role-based permissions

## API

GraphQL API provides:
- Flexible queries
- Real-time subscriptions
- Efficient data fetching
- Type-safe schema

## Developer Experience

- CLI for scaffolding
- Plugin system
- Webhook support
- REST API alternative
- SDK for popular frameworks

## Built With

- Node.js and Express
- GraphQL and Apollo Server
- PostgreSQL database
- Redis caching
- AWS S3 for media storage

## Use Cases

- Personal blogs
- Company blogs
- Documentation sites
- News websites
- Magazine platforms

Powerful enough for enterprises, simple enough for individuals.
