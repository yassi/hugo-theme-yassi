A modern task management application built for teams and individuals.

## Features

- Create and organize tasks in projects
- Real-time collaboration with WebSocket
- Drag-and-drop task organization
- Due dates and reminders
- File attachments
- Activity timeline
- Team member assignments
- Custom labels and filters

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Real-time**: Socket.io
- **Authentication**: JWT

## Key Highlights

### Real-time Updates

Tasks update instantly across all connected clients using WebSocket technology.

### Intuitive UI

Drag-and-drop interface makes task organization effortless.

### Mobile Responsive

Works seamlessly on desktop, tablet, and mobile devices.
